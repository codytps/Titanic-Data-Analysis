# Titanic-Data-Analysis
Titanic survival analysis using Python and data visualization.

### Project Overview

This project analyzes the Titanic passenger dataset from Kaggle, exploring how survival rates varied by passenger class, gender, and age. The goal is to identify factors that influenced survival and demonstrate data cleaning, visualization, and interpretation skills.

### Dataset

Source: Kaggle Titanic Dataset

Contents: 891 passengers, 12 variables including demographics, travel details, and survival status.

### Methods

Data cleaning and preparation using tidyverse in R

Exploratory Data Analysis (EDA)

Visualization of survival trends by class, gender, and age

Summary tables and bar/histogram plots to communicate findings

### Key Findings

Passenger Class: First-class passengers had the highest survival rate (~63%), third-class the lowest (~24%).

Gender: Women had a much higher survival rate (~74%) compared to men (~19%).

Age: Children under 10 had higher survival rates than adults.

### Tools Used

Programming & Libraries: R, tidyverse (dplyr, ggplot2, tidyr)

Visualization: ggplot2

Analysis: Data cleaning, EDA, statistical summaries
